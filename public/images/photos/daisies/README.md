# Daisy Stock Photos

## Required Images

Download these FREE stock photos and place them in this directory:

### 1. Hero Background (`hero-daisies.jpg`)
- **Source**: [Unsplash - Daisy Field](https://unsplash.com/s/photos/daisy-field)
- **Recommended**: Wide shot of white daisies in a field
- **Dimensions**: 1920x1080px minimum
- **File name**: `hero-daisies.jpg`

### 2. Story Section (`story-daisies.jpg`)
- **Source**: [Pexels - Wild Daisies](https://www.pexels.com/search/wild%20daisies/)
- **Recommended**: Close-up of daisies with soft focus background
- **Dimensions**: 1200x800px minimum
- **File name**: `story-daisies.jpg`

### 3. Programs Background (`programs-daisies.jpg`)
- **Source**: [Unsplash - Daisy Meadow](https://unsplash.com/s/photos/daisy-meadow)
- **Recommended**: Daisy meadow with sunlight
- **Dimensions**: 1920x1080px minimum
- **File name**: `programs-daisies.jpg`

### 4. Board Section (`board-daisies.jpg`)
- **Source**: [Pixabay - Daisies](https://pixabay.com/images/search/daisy/)
- **Recommended**: Elegant daisy bouquet or cluster
- **Dimensions**: 1200x800px minimum
- **File name**: `board-daisies.jpg`

## Quick Download Instructions

1. Visit the source links above
2. Search for images matching the description
3. Download in high resolution (free license)
4. Rename files to match the names above
5. Place in this directory
