# Nonprofit Standards

- Show donation allocation transparently (78% programs, 14% ops, 8% fundraising)
- Use real, database-driven impact metrics — never hardcode or exaggerate
- CTA hierarchy: Donate (primary) > Volunteer/Contact > Newsletter
- 3 clicks max from homepage to Stripe Checkout — no account creation required
- Mobile-first: tap targets >= 44px, body text >= 16px, stack tiers vertically
- Use Next.js `<Image>`, never raw `<img>` — lazy-load below fold
- All payments via Stripe Checkout (never handle card details server-side)
- Content voice: warm, authentic, hopeful — use "we/our", active voice, real numbers

Full reference: `docs/standards-nonprofit.md`
